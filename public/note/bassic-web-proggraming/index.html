    Belajar Terminal

    Linux adalah keluarga sistem operasi bebas dan sumber terbuka yang pada dasarnya dibangun di atas kernel Linux, yaitu sebuah kernel sistem operasi yang pertama kali dikembangkan oleh Linus Torvalds pada 1991. Linux dirilis di bawah Lisensi Publik Umum GNU versi 2. source : Wikipedia Nah di dalam OS linux inilah terdapat perintah linux, yang secara default terinstall ialah bash. Karena jaman dahulu OS linux tidak semudah sekarang, dimana untuk mengoperasikannya hanya menggunakan kode/diketik. Meskipun sekarang sudah ada gui, namun penggunaan code ini masih dibutuhkan, terutama dalam menjalankan sebuah aplikasi yang memang tidak memerlukan gui. Untuk lebih tahu bagaimana penggunaannya bisa tonton video dibawah (jika sudah ada). atau silahkan kunjungi link berikut
