    Web Desainer VS Web Developer, Penjelasan dan Perbedaan

     Hello world!

    Desain web dan web developer itu bedanya apa? Bagi kalian yang baru terjun di bidang IT pasti akan bingung dengan istilah web desainer dan web developer. Tapi tenang kali ini saya akan bahas setuntas mungkin.


    Web Desainer

    Secara singkat web desainer itu mengacu pada tampilan web. Dimana biasanya menggunakan Adobe xd atau software lainnya untuk membuat rancangan desain dari sebuah web. Yang nantinya akan diserahkan ke web developer. 

    Seorang web desainer tak hanya membuat berdasarkan bagus dipandang saja, namun juga harus memikirkan segi fungsionalitasnya juga. 

    Bagi seorang desainer prinsip berikut harus dipenuhi, yaitu Balance(keseimbangan), kontras, penekanan, konsistensi, unity. Yang meliputi antara lain:

    Desainer harus jeli ketika menambahkan tata  warna, gelap terang, ukuran, tekstur, dan bentuk yang kontras. Dimana harus seimbang satu sama lainnya. Jangan sampai ada beberapa bagian yang penting tapi malah tidak kelihatan. Seperti contohnya : tombol beli harusnya mencolok agar pengunjung mudah menemukannya dan dapat mempengaruhi psikologis pengunjung. Jangan sampai juga tulisan yang ada di web tidak terlihat atau bikin sakit mata saat lama membaca.


    Desainer juga harus paham betul tujuan dibuatnya sebuah web. Dimana tata letak nantinya berpengaruh. Seperti saat kita membuat desain portofolio dengan toko online akan sangat berbeda. Dimana portofolio hanya berisi indentitas orang saja sedangkan toko online dibutuhkan lebih banyak informasi mengenai sebuah produk.


    Dan yang terakhir website harus memiliki komposisi yang pas secara keseluruhan.  Berdasarkan teori Gestalt, kesatuan berkaitan dengan bagaimana otak manusia mengatur informasi secara visual dengan mengelompokkan elemen ke dalam kategori.


    Baca juga mengenai Front End dan Back end 


    Web Developer

    Untuk web developer sendiri, mereka hanya menerima desain yang sudah dibuat dan memecah menjadi bagian-bagian kecil dan merangkainya menjadi sebuah website. Pengembang biasanya menggunakan sebuah bahasa pemrograman sebagai pendekatan.

    Seperti HTML sebagai markup dan CSS sebagai style(merancang dan membentuk). Jika ingin lebih interaktif lagi, bisa gunakan javascript sebagai bahasa pemrogramannya. Dan biasanya sebuah web menggunakan PHP sebagai halaman dinamis. Dan untuk menghubungakan artikel/halaman satu sama lainnya biasanya menggunakan sebuah link.


    Dan untuk memanajemen sebuah konten biasanya sebuah blog menggunakan CMS sebagai editornya. Contoh CMS ialah, Wordpress, Blogger, atau bisa membuat sendiri. Sehingga kita tak perlu repot-repot memikirkan sebuah syntax. Tinggal pakai. Bagi yang belum pernah memakai, CMS itu mirip seperti Word office.


    FYI : umumnya Kedua hal tersebut dipisah, namun ada juga orang yang dapat keduanya, namun biasanya salah satunya pasti jelek.


    Kesimpulan

    Jadi, kesimpulannya adalah web desainer itu tampilan sedangkan web developer itu yang mengaplikasikan desain tersebut menjadi sebuah web.
